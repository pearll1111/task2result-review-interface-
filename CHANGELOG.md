# Changelog

## [1.0.0] — 2026-03-15

### Added
- Start screen with candidate name input and topic selector
- Quiz screen — 10 questions per topic, no answer hints during quiz
- Skip question with zero mark penalty
- **Result Review Screen** (main feature):
  - Final score with +1 / −0.25 / 0 marking
  - Summary: Total / Correct / Wrong / Skipped cards
  - Per-question review cards (green = correct, red = wrong, yellow = skipped)
  - Toggle switch to show/hide correct answers
- 80+ hardcoded questions across 8 tech topics
- Fully offline — single HTML file, zero dependencies
- Dark UI: #0a0e1a background, #00d4ff cyan accent
- Responsive for mobile and desktop
- Space Grotesk + JetBrains Mono typography
