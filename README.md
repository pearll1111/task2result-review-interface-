📊 Result Review Interface

An interactive **quiz result review system** built with pure HTML, CSS, and JavaScript.  
Candidates take a tech quiz and get a **detailed result review** at the end — showing scores, correct answers, and per-question breakdown.

---

## 🚀 Features

- 🧠 **80+ handcrafted questions** across 8 tech topics
- 👤 **Candidate name entry** before starting the quiz
- 🔒 **No answer reveal mid-quiz** — results shown only at the end
- 📊 **Detailed result review** with correct/wrong breakdown
- ➕ **Scoring system** — +1 for correct, −0.25 for wrong, 0 for skipped
- 📱 **Fully responsive** — works on mobile and desktop
- ⚡ **Zero dependencies** — no npm, no API, no internet needed

---
📝 Quiz
- Candidate name entry (required)
- 8 tech topics with 10 questions each
- No right/wrong feedback during quiz — all revealed at end

## 🖥️ How to Run

### Option 1 — Just double-click
Open `index.html` directly in your browser. No setup needed.

### Option 2 — VS Code Live Server
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

### Option 3 — Terminal
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```
Then open `http://localhost:8080` in your browser.

## 🎮 How to Play

1. Enter your **name**
2. Select a **topic** from the dropdown
3. Click **Start Quiz**
4. Choose your answer and click **Confirm Answer**
5. Use **Skip** to skip a question (no marks deducted)
6. After all 10 questions, click **See Results**
7. Review your answers with correct answers highlighted
